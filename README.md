# 📄 AI Resume Analyzer

An AI-powered resume analysis tool that extracts resume content, analyzes skills, calculates ATS score, and provides improvement suggestions.

## Features

- Resume PDF Upload
- ATS Score Calculation
- Skill Categorization
- Resume Section Analysis
- Skill Distribution Chart
- PDF ATS Report Download
- Resume Improvement Suggestions

## Tech Stack

- Python
- Streamlit
- PyPDF2
- Plotly
- ReportLab
